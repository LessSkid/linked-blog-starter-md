### 基本信息
```
世界名称: 妄想联盟 (Delusion Alliance)
性质: 现实与幻想交融的心理治疗空间
核心概念: 将心理防御机制具现化的治疗领域
独特性: 每个人的妄想都能在此获得理解和治疗
```

### 妄想联盟的世界结构
```
基本形态:
  - 类似现代城市，但建筑会根据访客心理状态变化
  - 街道布局反映不同类型的心理防御机制
  - 建筑风格混合现实与想象，理性与非理性
  - 天空呈现不断变化的心理色彩

区域划分:
  防御机制区: 各种心理防御的具现化空间
  记忆重构区: 被修改或美化的回忆展示区
  理想自我区: 人们想象中完美自己的居住地
  替代现实区: 逃避现实而构建的平行世界
  治愈中心区: Theo进行深度治疗的核心区域
```
#### 特殊居民 - 妄想人格
```
妄想人格的本质:
  - 不是真实的人，而是心理防御机制的拟人化
  - 代表不同类型的自我欺骗和心理保护
  - 每个妄想人格都有其存在的合理性和功能
  - 它们帮助真实的人类度过心理创伤

常见妄想人格类型:
  完美主义者: 认为自己无所不能的人格
  受害者: 将所有问题归咎于外界的人格
  救世主: 认为自己必须拯救所有人的人格
  隐形人: 相信自己不被任何人注意的人格
  时间旅行者: 活在过去或未来而非当下的人格
  批评家: 通过贬低他人来保护自己的人格
```

### 妄想联盟的设施与功能
#### 核心设施
```
妄想登记处:
  - 新来访者必须先"登记"自己的主要妄想
  - 工作人员是经验丰富的妄想人格
  - 没有判断，只有理解和分类
  - 为每个妄想分配合适的"居住区域"

现实检验中心:
  - 提供温和的现实测试服务
  - 不强制打破妄想，而是帮助理解其功能
  - 使用"现实镜"展示真实情况
  - 配备"缓冲室"避免现实冲击过大

重构工作坊:
  - 帮助访客重新构建更健康的认知
  - 与妄想人格对话，理解其保护功能
  - 逐步调整而非彻底推翻原有认知
  - 创造新的、更适应的心理防御

联盟议会厅:
  - 不同妄想人格间的协商和对话空间
  - 解决妄想间的冲突和矛盾
  - Theo主持的集体治疗会议
  - 学习如何让不同防御机制和谐共存
```
#### 特殊治疗空间
```
记忆编辑室:
  - 不是删除痛苦记忆，而是改变对它的感受
  - 帮助重新叙述人生故事
  - 发现痛苦经历中的意义和成长
  - 将创伤转化为智慧和力量

情感实验室:
  - 安全地体验和表达被压抑的情感
  - 妄想人格协助进行情感释放
  - 学习健康的情感处理方式
  - 理解情感与妄想的关系

身份试衣间:
  - 尝试不同的自我认同和角色
  - 体验"如果我是..."的各种可能性
  - 发现真实自我与理想自我的平衡点
  - 练习在现实中展现不同面向

关系剧场:
  - 重现和改写人际关系模式
  - 与妄想人格演练困难对话
  - 理解人际冲突中的投射和防御
  - 学习更健康的关系互动方式
```
